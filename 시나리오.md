# 한국콘텐츠기업협회 회원관리 프로그램 기획안

## 1. 프로젝트 개요
- **앱 이름**: 한국콘텐츠기업협회 회원관리 시스템
- **개발 환경**: Electron (macOS 전용, DMG 배포)
- **사용자**: 관리자 1인 전용 (회원가입 불필요)
- **회원 규모**: 약 100명

---

## 2. 회원 정보 항목
회원 1인당 관리하는 데이터 항목:
- 이름
- 연락처
- 이메일
- 가입일
- 회비 납부 여부
- 회비 납부 금액 (연간 단위)
- 메모 / 특이사항

---

## 3. 주요 기능

### 3-1. 회원 목록
- 테이블(표) 형식으로 전체 회원 나열
- 행 클릭 시 상세 정보 화면으로 이동

### 3-2. 회원 검색 및 필터
- 이름으로 검색
- 가입 연도별 필터
- 납부 상태별 필터 (납부 완료 / 미납)
- 위 3가지 조건을 복합 적용 가능

### 3-3. 회비 관리
- 연간 단위로 납부 여부 및 납부 금액 기록
- 미납 회원 별도 조회 가능

### 3-4. 회원 추가 / 수정 / 삭제
- 회원 정보 추가 및 수정 (저장 버튼으로 확정)
- 삭제 시 확인 팝업 표시 ("정말 삭제하시겠습니까?")

### 3-5. 통계 대시보드
- 총 회원 수
- 연간 납부 완료 회원 수 및 납부율
- 미납 회원 수

---

## 4. UI / 디자인 방향
- 색상 테마: 파란색 계열, 깔끔하고 단정한 느낌
- 앱 상단에 협회명 "한국콘텐츠기업협회" 표시

---

## 5. 제외 기능
- 회원가입 / 로그인 기능 없음
- 엑셀 / CSV 내보내기 및 가져오기 없음
- 인쇄 기능 없음

---

## 6. 기술 스택 (예정)
- **프레임워크**: Electron
- **UI**: HTML / CSS / JavaScript (또는 React)
- **데이터 저장**: SQLite (로컬 DB)
- **배포**: macOS DMG
