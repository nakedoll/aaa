# 한국콘텐츠기업협회 회원관리 프로그램 기획안

## 1. 프로젝트 개요
- **앱 이름**: 한국콘텐츠기업협회 회원관리 시스템
- **개발 환경**: Electron (macOS 전용, DMG 배포)
- **사용자**: 관리자 1인 전용 (회원가입 / 로그인 불필요)
- **회원 규모**: 약 100개 기업

---

## 2. 회원 정보 항목 (기업 단위)
회원 1개 기업당 관리하는 데이터 항목:
- 회사명
- 대표자 이름
- 담당자 이름
- 담당자 연락처
- 담당자 이메일
- 가입일
- 회원 상태 (활동 / 휴면 / 탈퇴)
- 메모 / 특이사항

---

## 3. 데이터베이스 구조

### members (회원 테이블)
| 컬럼 | 설명 |
|---|---|
| id | 고유 식별자 |
| 회사명 | |
| 대표자 이름 | |
| 담당자 이름 | |
| 담당자 연락처 | |
| 담당자 이메일 | |
| 가입일 | |
| 회원 상태 | 활동 / 휴면 / 탈퇴 |
| 메모 | |

### dues (회비 테이블)
| 컬럼 | 설명 |
|---|---|
| id | 고유 식별자 |
| member_id | 회원 외래키 |
| 연도 | 납부 연도 |
| 납부금액 | |
| 납부일 | |
| 상태 | 납부완료 / 미납 |
- member_id + 연도는 중복 불가
- 납부는 전액 일시 납부만 허용

### dues_settings (연도별 기준 회비 테이블)
| 컬럼 | 설명 |
|---|---|
| 연도 | |
| 기준금액 | 해당 연도 기준 회비 금액 |
- 기준 회비는 연도별로 변경 가능
- 기업별 회비 금액은 동일

## 4. 주요 기능

### 4-1. 회원 목록
- 테이블(표) 형식으로 전체 회원 나열
- 행 클릭 시 상세 정보 화면으로 이동
- 기본 표시 열: 회사명, 대표자, 담당자, 연락처, 회원 상태, 납부 상태

### 4-2. 회원 검색 및 필터
- 회사명으로 검색
- 가입 연도별 필터
- 납부 상태별 필터 (납부 완료 / 미납)
- 위 3가지 조건 복합 적용 가능

### 4-3. 회비 관리
- 연간 단위, 전액 일시 납부
- 연도별 기준 회비 금액 설정 (매년 변경 가능)
- 미납 회원 별도 조회 가능
- 납부 완료 여부는 수동으로 등록

### 4-4. 회원 추가 / 수정 / 탈퇴 처리
- 회원 정보 추가 및 수정 (저장 버튼으로 확정)
- 삭제 대신 탈퇴 처리 (데이터 보존, 과거 회비 이력 유지)
- 탈퇴 처리 시 확인 팝업 표시
- 탈퇴 회원은 목록에서 필터로 숨기거나 표시 가능

### 4-5. 통계 대시보드
- 총 회원 수 (활동 / 휴면 / 탈퇴 구분)
- 연간 납부 완료 회원 수 및 납부율
- 미납 회원 수

### 4-6. 백업
- 앱 실행 시 자동 백업 (최근 5개 보관)
- DB는 Electron 사용자 데이터 디렉터리 하위에 저장
- 백업 파일에서 복원 기능 포함

---

## 5. UI / 디자인 방향
- 색상 테마: 파란색 계열, 깔끔하고 단정한 느낌
- 앱 상단에 협회명 "한국콘텐츠기업협회" 표시

---

## 6. 제외 기능
- 회원가입 / 로그인 기능 없음
- 엑셀 / CSV 내보내기 및 가져오기 없음 (최초 데이터는 수동 입력)
- 인쇄 기능 없음
- 부분 납부 없음 (전액 일시 납부만)

---

## 7. 기술 스택
- **프레임워크**: Electron
- **UI**: HTML / CSS / JavaScript (또는 React)
- **데이터 저장**: SQLite (로컬 DB)
- **보안**: nodeIntegration: false, contextIsolation: true, IPC 통신으로 DB 접근
- **배포**: macOS DMG
